Tools around graphs

titbit.graphs.attribute_dependency_graph(cls)[source]

Return a graphviz Digraph of the attribute dependencies of a class.

An attribute (method or property) is dependent on another attribute if it uses it in its definition.

titbit.graphs.graph_node_ids(graph) → set[source]

Return the node identifiers (labels/IDs) of a Graphviz Digraph.

WARNING: This is a fragile method that relies on the structure of the graphviz Digraph object. I could see no other way to do it in graphviz’s API. See: https://graphviz.readthedocs.io/en/stable/api.html

Parameters:

graph (graphviz.graphs.Digraph) – The graph instance.

>>> import graphviz
>>> g = graphviz.Digraph('G', filename='hello.gv')
>>> g.node('A')
>>> g.node('B')
>>> g.node('C')
>>> g.edges(['AB', 'BC', 'CA'])
>>> sorted(graph_node_ids(g))
['A', 'B', 'C']
titbit.graphs.mermaid_to_graphviz(mermaid_code, extra_replacements=(), *, prefix='', suffix='', egress=None)[source]

Converts mermaid code to graphviz code.

>>> mermaid_code = '''
... graph TD
... A --> B & C
... B & C --> D
... '''
>>> graphviz_code = mermaid_to_graphviz(mermaid_code)
>>> print(graphviz_code)  
digraph G {
    A -> B , C
    B , C -> D
}

>>> mermaid_code = '''
... graph TD
...     A[Score] --> B[Part]
...     B --> C[Measure]
... '''
>>> print(mermaid_to_graphviz(mermaid_code))
digraph G {
    A [label="Score"]
    B [label="Part"]
    A -> B
    C [label="Measure"]
    B -> C
}
titbit.graphs.update_node_attributes(graph, node_ids: Iterable, attributes: dict | None = None, *, shape: str | None = None, fillcolor: str | None = None, style: str | None = None, **extra_attributes)[source]

Update attributes for specific nodes in a Graphviz Digraph.

See list of all node attributes here: https://graphviz.org/docs/nodes/

Parameters:

  • graph (graphviz.graphs.Digraph) – The graph instance.

  • node_ids – A list of node identifiers (labels/IDs) to update.

  • shape – The shape attribute to update for the nodes.

  • attributes – A dictionary of attributes to apply to the nodes.

  • fillcolor – Color of interior of node. (Required: style=’filled’)

  • style – Set style information for components of the graph.

  • extra_attributes – Additional attributes to apply to the nodes.

>>> import graphviz
>>> g = graphviz.Digraph('G', filename='hello.gv')
>>> g.node('A')
>>> g.node('B')
>>> g.node('C')
>>> g.edges(['AB', 'BC', 'CA'])
>>> g = update_node_attributes(g, ['A', 'C'], color='red', shape='box')
>>> print(g.source)  
digraph G {
    A
    B
    C
    A -> B
    B -> C
    C -> A
    A [color=red shape=box]
    C [color=red shape=box]
}
